# @clawrun/runtime

Sandbox lifecycle and orchestration for ClawRun. Manages sandbox start, heartbeat keep-alive, snapshot/stop, and wake-on-message. Includes the sidecar process (supervisor, heartbeat, health) that runs inside the sandbox, and a Redis-backed state store.

**Docs:** [clawrun.sh/docs](https://clawrun.sh/docs)
