---
title: SDK
description: Programmatic TypeScript API for deploying, managing, and interacting with ClawRun agent instances.
---

The `@clawrun/sdk` package provides a TypeScript API for everything the CLI does, plus streaming chat, thread management, and sandbox operations. Use it to integrate ClawRun into your own applications, build custom dashboards, or automate deployments.

## Install

```bash
npm install @clawrun/sdk
```

## Quick example

```typescript
import { ClawRunClient } from "@clawrun/sdk";

const client = new ClawRunClient();
const instance = client.connect("https://my-agent.vercel.app", "jwt-secret");

// Send a message and get the full response
const response = await instance.sendMessage("What can you do?");
console.log(response.parts);

// Check agent health
const health = await instance.health();
console.log(health.status); // "ok"
```
