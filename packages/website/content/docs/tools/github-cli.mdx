The official [GitHub CLI](https://github.com/cli/cli) (`gh`) for managing repositories, issues, and pull requests from inside the sandbox.

| Property | Value             |
| -------- | ----------------- |
| ID       | `gh-cli`          |
| Version  | `2.65.0`          |
| Binary   | `~/.local/bin/gh` |

## Installation

Downloaded directly from GitHub releases as a `.tar.gz` archive. No additional post-install steps required.

**Required network domains:**

- `github.com`
- `release-assets.githubusercontent.com`
- `api.github.com`

If your sandbox uses a restricted [network policy](/docs/getting-started/configuration), the deploy CLI will prompt you to add these domains to the allowlist.

## Skill file

GitHub CLI ships a SKILL.md that is seeded into the agent's workspace at `workspace/skills/gh-cli/SKILL.md` during deploy. It teaches the agent common `gh` workflows for issues, PRs, and repository management.
