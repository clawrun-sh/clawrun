# @clawrun/provider

Abstract `SandboxProvider` interface for ClawRun. Defines the contract for sandbox runtimes (create, get, list, snapshot, delete) and snapshot retention policies. Provider implementations live in separate packages like `@clawrun/provider-vercel`.

**Docs:** [clawrun.sh/docs](https://clawrun.sh/docs)
