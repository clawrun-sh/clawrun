ClawRun acts as a translation layer between **agents** and **providers**. Both interfaces are pluggable, so you're never locked into a single runtime or a single agent.

## Agents

ClawRun's `Agent` interface is pluggable. Implementations self-register via the agent registry.

| Agent                                             | Status      | Description                                                    |
| ------------------------------------------------- | ----------- | -------------------------------------------------------------- |
| [ZeroClaw](https://github.com/nichochar/zeroclaw) | Supported   | Fully autonomous AI agent with tools, memory, and cron support |
| OpenClaw                                          | Coming soon |                                                                |
| NanoClaw                                          | Coming soon |                                                                |

## Messaging channels

ClawRun supports 15 messaging channels. Channels with wake-hook adapters can wake a sleeping sandbox when a message arrives. Other channels are configured inside the agent and work when the sandbox is running.

### Wake-hook enabled

These channels can wake a sleeping sandbox when a message arrives:

| Channel       | ID         |
| ------------- | ---------- |
| Telegram      | `telegram` |
| Discord       | `discord`  |
| Slack         | `slack`    |
| WhatsApp      | `whatsapp` |
| Lark / Feishu | `lark`     |
| QQ            | `qq`       |

### Agent-side only

These channels are configured in the agent but do not have wake-hook adapters yet:

| Channel           | ID               |
| ----------------- | ---------------- |
| iMessage          | `imessage`       |
| Matrix            | `matrix`         |
| Signal            | `signal`         |
| Linq              | `linq`           |
| IRC               | `irc`            |
| DingTalk          | `dingtalk`       |
| Nostr             | `nostr`          |
| Nextcloud Talk    | `nextcloud_talk` |
| Webhook (generic) | `webhook`        |

## Tools

Tools are installed inside the sandbox at startup. Select which tools to enable during [`clawrun deploy`](/docs/cli/deploy).

| Tool          | ID              | Description                                                |
| ------------- | --------------- | ---------------------------------------------------------- |
| Agent Browser | `agent-browser` | Headless Chromium browser for web browsing and screenshots |
| GitHub CLI    | `gh-cli`        | GitHub CLI for managing repos, issues, and PRs             |
