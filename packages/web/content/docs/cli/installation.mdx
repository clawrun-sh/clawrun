## Requirements

- Node.js 20 or later

The [`deploy`](/docs/cli/deploy) command will also check for the [Vercel CLI](https://vercel.com/docs/cli) and attempt to install it automatically if missing. If you prefer to install it manually:

<PackageCommand cmd="npm install -g vercel" />

## Package managers

<PackageCommand cmd="npm install -g clawrun" />

Verify the installation:

```bash
clawrun --version
```

## Instance directory

ClawRun stores all instance data under `~/.clawrun/`. You can override this by setting the `CLAWRUN_HOME` environment variable:

```bash
export CLAWRUN_HOME=/path/to/custom/dir
```

## Updating

<PackageCommand cmd="npm install -g clawrun@latest" />
