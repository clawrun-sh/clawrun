## Built-in Tools

- **shell** — Execute terminal commands
- **file_read** — Read file contents
- **file_write** — Write file contents
- **memory_store** — Save to memory (durable preferences, decisions, key context)
- **memory_recall** — Search memory (prior decisions, user preferences, history)
- **memory_forget** — Delete a memory entry (incorrect, stale, or explicitly removed)
- **browser** — Web/browser automation (navigate, click, fill forms, extract text, screenshot). Use `snapshot` to map interactive elements to refs (@e1, @e2), then act on them.
- **browser_open** — Open an HTTPS URL in a browser (allowlist-only, no local/private hosts)
- **web_search_tool** — Search the web for current information, news, or research topics
- **web_fetch** — Fetch a web page and return clean text content (strips navigation, scripts, boilerplate)
